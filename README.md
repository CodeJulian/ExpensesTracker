# ExpensesTracker

Simple application made for tracking the daily expenses.

## What things you need to run it:

You have to install:
```
MySQL Connector/J 
```
and run this script to make a database
```
https://github.com/WhereIsMyCoffe/ExpensesTracker/blob/master/database.sql
```
